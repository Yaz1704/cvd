# CVD Classroom Games
Interactive activities for teaching **Cardiovascular Disease (CVD)** from a sociological perspective.

## Contents
- `healthy_heart_game.html` — Lifestyle choices mini-simulation (open in any browser).
- `sociology_matching_game.html` — Matching game for sociological theories & key ideas.
- `Healthy_Heart_Game_Slide.pptx` — PowerPoint slide that launches the lifestyle game.
- `Sociology_Matching_Game_Slide.pptx` — PowerPoint slide that launches the matching game.
- `Cardio_with_Game_Final.pptx` — Your presentation with the lifestyle game slide added.

## Run locally
Open any `.html` file in your browser (Chrome, Edge, Firefox, Safari). No install required.

## Use in PowerPoint
Keep the `.pptx` and its corresponding `.html` in the same folder. Click the button on the slide to launch the game.

## Upload to GitHub
Upload the contents of this folder or the ZIP to a repo. GitHub Pages can host the games directly.
